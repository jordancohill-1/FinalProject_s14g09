HERE'S HOW WE DID IT!

POSTER SCRAPING

With help from the following repository, movie_rating_prediction, about 5000 posters with corresponding movie data were scraped from a list of movie titles at The Numbers.

QUANTIZATION AND DOMINANT COLORS

Using Python's OpenCV library, movie posters were quantized into 4 clusters. From there, the quantized images were explored with the ColorThief library to identify the dominant colors.

TRAIN TEST

Once the dominant colors were identified, a train test algorithm was created with the SKLearn library to predict a movie's IMDB rating from the dominant colors in it's poster.

FACIAL DETECTION AND RECOGNITION

As an added bonus we used the facial_recognition library to find and identify faces. A pre-trained Convolutional Neural Network (CNN) identifies faces in the poster. These faces are then compared to the faces of 100 actors, stored in the database. If we found possible matches, each face would be compared to the face in the poster and the one with the smallest euclidian distance would be chosen.
